## Auto-Download
If you don't download any models manually, the application will automatically download:
- inswapper_128.onnx (best quality)
- GFPGANv1.4.pth (face enhancement)

## Model Priority
The application uses this priority order:
1. inswapper_128.onnx (full precision - best quality)
2. inswapper_128_fp16.onnx (half precision - faster, less VRAM)

## Hardware Recommendations
- **High-end GPU (RTX 3070+, 8GB+ VRAM)**: Use inswapper_128.onnx
- **Mid-range GPU (GTX 1660, RTX 3060, 4-6GB VRAM)**: Use inswapper_128_fp16.onnx
- **Low-end GPU (4GB or less VRAM)**: Use inswapper_128_fp16.onnx
